# Interstellarium

## Project installation and launch
#### 1. Clone repository
#### 2. Install Python
You should use Python 3.11.5 or other compatible versions
#### 3. Go to project folder 
#### 4. Install virtual enviroment
#### 5. Install requirements.txt
#### 6. Run application
Debug mode off:
```commandline
flask run
```
Debug mode on:
```commandline
flask run --debug
```
